## ‚ùì FAQ

**Q: Where do I put my profile picture?**
A: Save it as `ProfilePic.jpeg` (200√ó200px) in the `assets/` folder

**Q: How do I edit my name and info?**
A: Open `index.html` in VS Code and search for your current name

**Q: How do I change the colors?**
A: Edit `style.css` - look for `:root { --primary-color: #38bdf8; }`

**Q: How do I make emails work?**
A: Read [README.md](README.md) email configuration section

**Q: How do I deploy online?**
A: Follow [DEPLOYMENT.md](DEPLOYMENT.md) for your chosen platform

**Q: Why does my website say "Cannot GET /"?**
A: Make sure server is running (`npm run dev`) and you're at `localhost:3000`

**Q: Can I customize the design?**
A: Yes! All styling is in `style.css` - edit colors, fonts, sizes

**Q: What if something breaks?**
A: Check [CHECKLIST.md](CHECKLIST.md) troubleshooting section
